e-Sports calendar bot.

It posts events at their start time.

Software by <a href="https://metalhead.club/@mariusor">@mariusor</a>: <a href="https://git.sr.ht/~mariusor/othrys">https://git.sr.ht/~mariusor/othrys</a>.

The releases are based on data publicly available on the calendars from <a href="https://tl.net/">TeamLiquid</a> and <a href="https://www.plusforward.net/">PlusForward</a>.

Many thanks to all the volunteers that publish it.
